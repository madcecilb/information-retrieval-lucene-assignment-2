command to run program:
	java -jar <Assignment-2.jar location> <site to be indexed and searched> <query> 
<number of letters in excerpt> <number of excerpts>
for example: 
	java -jar C:\Assignment-2.jar http://www.techcrunch.com brand+new+device 100 2 
by default program searches for a query: 
	brand new device
on a site 
	http://www.techcrunch.com
so, command:
	java -jar C:\Assignment-2.jar
searches for:
	brand new device on a site http://www.techcrunch.com with 100 letters in excerpt with 2 exceprts
	
file locations:
	Assignment-2.jar inside folder /runnable_jar
	documentation inside folder /doc
